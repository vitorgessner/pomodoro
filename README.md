# üçÖ Pomodoro - project from roadmapsh

The project consists of implementing a timer for the Pomodoro technique. The initial timers are set to: 25 minutes for work/study sessions, 5 minutes for short breaks, and 15 minutes for long breaks after four work/study sessions. However, you can change these values according to your needs. The idea was taken from Roadmap.sh
